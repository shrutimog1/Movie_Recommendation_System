This project is a content-based movie recommendation system developed using Python and machine learning techniques.

## Features
- Recommends movies based on similarity
- Uses movie title and genre information
- Simple and efficient implementation

## Dataset
- movies.csv (MovieLens dataset)
- Columns: movieId, title, genres, etc.
Note: The dataset (movies.csv) is large, so GitHub does not preview it directly. Please **download the file to view** its contents.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Algorithm
- TF-IDF Vectorization
- Cosine Similarity

## How to Run
1. Open the notebook in Google Colab
2. Upload movies.csv
3. Run all cells
4. Enter a movie title to get recommendations
